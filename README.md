This repository contains a collection of quantitative finance projects
organized by core methodological domains. Each folder groups self-contained
projects that demonstrate modeling, risk analysis, and decision-making
skills used in quantitative finance roles.

- Machine Learning and Predictive Modeling
- Market Research
- Probabilistic Decision Models
- Quantitative Trading and Market Microstructure
- Risk Management and Portfolio Theory
- Stochastic Models and Derivatives
- Valuation Modeling
